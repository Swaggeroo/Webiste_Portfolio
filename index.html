AutoChannelBot

It's a Discord bot that ensures that there is always a free voice channel in the server. If someone joins a channel that is declared to be always available, a new one will be created for future users. If the last user leaves a channel, the bot will check if there is any other free one od the same type and if so, it will delete the current one.

AndroidWirelessADB-PowerShell

This is a PowerShell script that was created to connect to an Android device wireless via ADB. You just had to connect you device once per USB-Port, and then you can use the script to connect it wireless. Now it's kinda outdated, because if you use a new Android-Device, you can do the same thing just in the Dev-Options. But it's still usefully for older devices.
